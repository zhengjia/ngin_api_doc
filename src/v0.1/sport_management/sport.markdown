Sport
=====

Description
-----------
Get all available sports data

Actions
**INDEX** `GET /sports`
 
Attributes
----------
**id** *(integer)*	sport id assigned by NGIN
**name**	*(String)*	sport name

Example Requests
----------------
    curl -H "NGIN-API-TOKEN: your_api_token" -H "NGIN-API-VERSION: 0.1"  "https://api.api-sandbox.ngin.com/sports"
